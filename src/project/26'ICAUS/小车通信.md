## 串口通信
下面是以后使用串口直接进入小车 Jetson 终端的说明书。

**连接方式**
1. USB 串口线接电脑。
2. Windows 设备管理器里确认串口号，一般是：
   ```text
   COM5
   ```
3. Jetson 端对应串口是：
   ```text
   /dev/ttyUSB0
   ```

**PuTTY 设置**
打开 PuTTY，选择 `Session`：

```text
Connection type: Serial
Serial line: COM5
Speed: 57600
```

然后左侧进 `Connection > Serial`，确认：

```text
Data bits: 8
Stop bits: 1
Parity: None
Flow control: None
```

点击 `Open`。

**进入终端**
打开后如果是黑屏，按一下 `Enter`。

现在已经配置成自动登录，正常会直接看到：

```text
jetson@yahboom:~$
```

看到这个就可以直接输入 Linux 命令控制小车。

**如果又出现登录提示**
如果看到：

```text
yahboom login:
```

注意 `yahboom` 是主机名，不是账户名。

手动登录用：

```text
用户名：jetson
密码：yahboom
```

**如果输入时看不到字**
这是 PuTTY 本地回显没开，不影响发送。想看到自己输入的字：

1. 右键 PuTTY 窗口标题栏
2. `Change Settings...`
3. 左侧选 `Terminal`
4. `Local echo` 改成 `Force on`
5. `Local line editing` 改成 `Force off`
6. 点 `Apply`

**如果 PuTTY 连不上**
如果提示打不开 `COM5`，通常是串口被占用了。

处理办法：

```text
关闭其他 PuTTY / 串口助手 / Arduino IDE / CuteCom
重新打开 PuTTY
```

也可以在 PowerShell 查看串口是否存在：

```powershell
mode COM5
```

**Jetson 端当前配置**
我已经在 Jetson 上配置了串口终端服务：

```text
serial-getty@ttyUSB0.service
57600 8N1
自动登录 jetson
开机自动启用
```

正常情况下以后不用再配置 Jetson，只要插线，用 PuTTY 连 `COM5` 即可。
