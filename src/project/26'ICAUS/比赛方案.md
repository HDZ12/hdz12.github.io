## 方案一
可以做到，而且已经给你装好了。

小车端目录：

```text
/home/jetson/race_maze
```

Windows 端脚本：

```text
D:\html\race_control_com5.ps1
D:\html\run_race_control_com5.bat
```

我也已经把旧的串口控制/底盘进程清干净了，当前小车比赛栈是停止状态。

**测试流程**

1. SSH 进 Jetson，启动建图 + 底盘 + 雷达：

```bash
race_start
```

2. 打开比赛串口控制/记录：

```bash
race_drive_serial_on
```

3. 关闭 PuTTY 串口窗口，在 Windows 运行：

```powershell
cd D:\html
.\race_control_com5.ps1
```

或直接运行：

```text
D:\html\run_race_control_com5.bat
```

4. 第一遍手动开迷宫：

```text
q/w/e/a/s/d/z/x 控车
空格 停止
m 标记救援点
p 标记终点
1-6 调速
```

程序会自动记录路线，不需要每个弯手动打点。

5. 建完图后保存地图：

```bash
race_save
```

6. 把小车放回同一起点、同一朝向，自动按记录路线复跑：

```bash
race_run_replay
```

常用检查：

```bash
race_status
```

停止全部：

```bash
race_stop
```

第一轮建议把速度放低、场地留空，先验证“记录路线 -> 复跑”能走通，再去迷宫里调。
