## 题目链接
[860. 柠檬水找零](https://leetcode.cn/problems/lemonade-change/description/)
## 解题思路
值改变5和10的状态。
## 代码实现
```java
class Solution {
    public boolean lemonadeChange(int[] bills) {
        int five = 0;
        int ten = 0;

        for (int bill : bills) {
            if (bill == 5) {
                five++;
            } else if (bill == 10) {
                if (five == 0) {
                    return false;
                }
                five--;
                ten++;
            } else {
                if (ten > 0 && five > 0) {
                    ten--;
                    five--;
                } else if (five >= 3) {
                    five -= 3;
                } else {
                    return false;
                }
            }
        }

        return true;
    }
}
```
## 复杂度
时间复杂度: $O(n)$
空间复杂度: $O(1)$
