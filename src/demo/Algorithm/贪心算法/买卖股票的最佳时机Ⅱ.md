## 题目链接
[122.买卖股票的最佳时机||](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-ii/description/)
# 解题思路
只取利益为正的。
# 代码实现
```java
class Solution {
    public int maxProfit(int[] prices) {
        int profit = 0;

        for (int i = 1; i < prices.length; i++) {
            if (prices[i] > prices[i - 1]) {
                profit += prices[i] - prices[i - 1];
            }
        }

        return profit;
    }
}
```
# 复杂度
时间复杂度: $O(n)$
空间复杂度: $O(1)$
