# 题目链接
[455. 分发饼干](https://leetcode.cn/problems/assign-cookies/description/)
# 解题思路
两个快排，每次选择最少的。
# 代码实现
```java
class Solution {
    public int findContentChildren(int[] g, int[] s) {
        Arrays.sort(g);
        Arrays.sort(s);
        int i=0;
        int j=0;
        int count = 0;
        while (i<g.length && j<s.length) {
            if (g[i]<=s[j]) {
                i++;
                j++;
                count++;
            }
            else {
                j++;
            }
        }
        return count;
    }
}
```
# 复杂度
时间复杂度: $O(nlogn)$
空间复杂度: $O(1)$
