# 题目链接
[376.摆动序列](https://leetcode.cn/problems/wiggle-subsequence/description/)

# 解题思路
只放波峰和波谷。
# 代码实现
```java
class Solution {
    public int wiggleMaxLength(int[] nums) {
        int n = nums.length;
        if (n <= 1) {
            return n;
        }

        int count = 1;
        int preDiff = 0;

        for (int i = 1; i < n; i++) {
            int curDiff = nums[i] - nums[i - 1];

            if ((preDiff <= 0 && curDiff > 0) || 
                (preDiff >= 0 && curDiff < 0)) {
                count++;
                preDiff = curDiff;
            }
        }

        return count;
    }
}
```
# 复杂度
时间复杂度: $O(n)$
空间复杂度: $O(1)$
